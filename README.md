# AI Trip Planner

AI Trip Planner is a web-based application that generates personalized travel itineraries using AI.  
It simplifies travel planning by automating destination selection, activity planning, and route visualization based on user preferences.

---

## Project Overview

Traditional travel planning requires visiting multiple websites, comparing options, and manually organizing information, which is time-consuming and overwhelming.  
AI Trip Planner solves this problem by using AI to generate customized travel plans based on inputs such as destination, budget, duration, and interests.

---

## Objectives

- Automate trip itinerary generation using AI  
- Provide a responsive and user-friendly interface  
- Securely store user preferences and itineraries  
- Integrate maps for destination and route visualization  
- Reduce manual planning effort and save time  

---

## Key Features

- AI-powered personalized itinerary generation  
- Interactive and responsive web interface  
- Secure data storage and authentication  
- Route and destination visualization using maps  
- Simple and efficient travel planning experience  

---

## Technologies Used

- Frontend: React.js, Vite  
- Styling: Tailwind CSS  
- Backend & Authentication: Firebase  
- AI Integration: Gemini AI  
- Maps Integration: Google Maps API  

---

## System Flow

1. User enters trip preferences such as destination, budget, duration, and interests  
2. AI processes the inputs and generates a customized itinerary  
3. User data and itineraries are securely stored in Firebase  
4. The generated itinerary and routes are displayed on the web interface
